 Descripción

Este repositorio contiene dos proyectos desarrollados en C# como parte del Desafío 2. Cada programa implementa una funcionalidad independiente ejecutable desde la consola.

 Proyectos

- **`JuegoAhorcado.cs`** — Juego de adivinanza de palabras con sistema de categorías, pistas y control de intentos.
- **`SistemaNotas.cs`** — Sistema de registro de estudiantes con cálculo de promedios y clasificación de calificaciones.

 Tecnologías

- Lenguaje: C#
- Plataforma: .NET (aplicación de consola)
- IDE recomendado: Visual Studio Code

 Cómo ejecutar

1. Clonar o descargar el repositorio.
2. Abrir el archivo `.sln` o el `.cs` correspondiente en Visual Studio.
3. Seleccionar el proyecto a ejecutar como proyecto de inicio.
4. Presionar `F5` para compilar y ejecutar.
